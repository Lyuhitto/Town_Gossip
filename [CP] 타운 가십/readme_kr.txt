안녕하세요 Tell Everyone(Town Gossip)모드 한국어 번역입니다.
이 번역을 이용해주셔서 감사합니다!
해당 모드의 1.0.1 버전에 맞게 실행됩니다.

각종 제보는
1. 넥서스 메신저
2. 해당 번역의 포스트
3. 네이버 스타듀 밸리 카페 쪽지 (류히또)
4. github에 이슈 올리기
로 부탁드립니다.

==사용법==
1. 원본 모드를 설치합니다.
2. 업로드된 압축 파일을 받습니다.
3. 압축을 풀고 내용물을 원본 폴더 안에 덮어 씌웁니다.
